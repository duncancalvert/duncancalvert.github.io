BLEU
  • BLEU (BiLingual Evaluation Understudy) is a performance evaluation metric originally developed for machine-translated text from one language to another. Specifically, it calculates the precision for each matching Ngram between an LLM output and the expected output. It uses this to calculate their geometric mean and applies a brevity penalty if needed
